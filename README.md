# Shortest Path Visualizer

KU Campus Road Network | COMP 323 Graph Theory | Kathmandu University | May 2026

## Overview

This project implements and compares Dijkstra's Algorithm and Bellman-Ford Algorithm on a weighted
graph modeled from the Kathmandu University campus road network, with a web-based visualization.

## Goals

- Implement both algorithms from scratch (no built-in shortest path helpers).
- Ensure both algorithms return identical paths and distances.
- Provide a Flask API and a simple frontend visualization.

## Project Phases 

1. Project setup and structure
2. Campus graph data (JSON)
3. Graph builder (adjacency list + edge list)
4. Dijkstra implementation and unit tests
5. Bellman-Ford implementation and unit tests
6. Flask API
7. Frontend visualization
8. Final validation

## Notes

- All node names come from the University graph data file.
